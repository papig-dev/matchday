# Matchday

Browser-based football match viewer. Watch simulated league matches with 2D pitch visualization.

**All team names, player names, and statistics are entirely fictional and for entertainment purposes only. Not affiliated with or endorsed by any real football club, league, or organization.**

## Tech

Pure frontend — single HTML file. HTML5 Canvas + Vanilla JS. Deploy via Vercel.

```bash
cd frontend && npx vercel --prod
```
